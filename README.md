# Tensor-26-Syntax-Squad-
 LLM-Powered Multilingual Support Ticket Classification and Routing System
 🌍 Overview

This project is an intelligent customer support system that classifies and routes support tickets using Natural Language Processing techniques.

It can process tickets in multiple languages, analyze sentiment, detect urgency, and automatically assign them to the appropriate team.

✨ Features
🌐 Multilingual ticket processing
🧠 Rule-based + NLP classification
😊 Sentiment analysis using TextBlob
⚡ Urgency detection (High / Medium / Low)
🏢 Automatic team routing
📊 Batch processing using CSV upload
🎛️ Interactive UI using Streamlit
